# Сайт для направления СИП-домов

## Что собрано

Статический сайт для оффера Baraboy Evgen: домокомплекты из СИП-панелей под заказ в Краснодаре.
Текущая версия перестроена вокруг заявки на подготовку проекта дома.

Основной оффер:

- СИП-дом 56 м²;
- размер 7 × 8 м;
- 1 этаж;
- свайный фундамент;
- цена из объявления: 1 400 000 ₽;
- ориентир: 25 000 ₽/м²;
- состав: фундамент, обвязка, пол, стены, перегородки, перекрытия, кровля, терраса или крыльцо, окна, входная дверь, коммуникации, договор, гарантия, проект в подарок.

## Файлы

- `index.html` — структура сайта и текст.
- `styles.css` — адаптивная верстка с мобильным квизом.
- `script.js` — шапка при скролле, мобильное меню, пошаговый квиз, калькулятор стоимости и подготовка заявки.
- `assets/` — демонстрационные изображения для макета, в JPEG для веб-загрузки.

## Источник данных

Текст и параметры взяты из объявления Avito, которое пользователь прислал в чат:

- продавец: Baraboy Evgen;
- город: Краснодар;
- объявление: домокомплект из СИП-панелей с фундаментом;
- дата объявления: 30 мая 2026;
- номер объявления: 8165284654.

## Текущая логика заявки

На сайте есть мобильный 4-шаговый квиз для первичной заявки на подготовку проекта:

- тип дома через карточки с изображениями;
- площадь и статус участка через быстрые кнопки;
- желаемая комплектация с базовыми выбранными пунктами;
- телефон, имя и удобный канал связи в финальном шаге.

После заполнения сайт формирует текст заявки. В прототипе WhatsApp открывает
подготовленное сообщение, а для Max текст заявки копируется в буфер обмена.
Прямая отправка подключается после выбора рабочего номера и канала приема заявок.
